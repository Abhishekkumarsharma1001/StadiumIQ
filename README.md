# 🏟️ Smart Stadium Flow

A modern web application built using **React + TypeScript + Vite** to manage and visualize smart stadium operations efficiently.

---

## 🚀 Features

* ⚡ Fast performance with Vite
* 🧭 Routing with React Router
* 🔄 Data fetching using React Query
* 🎨 Clean UI with reusable components
* 🔔 Toast notifications system
* 📱 Responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React, TypeScript
* **Build Tool:** Vite
* **Routing:** React Router DOM
* **State/Data:** React Query
* **Styling:** Tailwind CSS

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/smart-stadium-flow.git
cd smart-stadium-flow
```

---

### 2️⃣ Install dependencies

```
npm install
```

---

### 3️⃣ Run the development server

```
npm run dev
```

---

### 4️⃣ Open in browser

```
http://localhost:5173/
```

---

## 📦 Build for Production

```
npm run build
```
